**Course:** Course 1 — Frontend Web Development

## Project Overview
A dynamic image gallery built with HTML and CSS featuring filtering capabilities and a responsive layout. Users can filter images by category (Landscapes, Portraits, Architecture) with smooth CSS transitions.

## Features
- Image grid with filter buttons
- Responsive layout (4 columns desktop, 2 tablet, 1 mobile)
- Hover effects on image cards
- Smooth transitions and filtering animations

## Tech Stack
- HTML5
- CSS3 (Grid and Flexbox)
